# OpenDih Website

Official website for OpenDih, a pioneering collective building the next generation of large language models.

## Running Locally

Simply serve the files with any static file server:

```bash
just live server it 🥀
```

Then open `http://localhost:8000` in your browser.

## Structure

```
website/
├── index.html      # Main page
├── style.css       # Styles
├── script.js       # Client-side JavaScript
└── src/
    ├── logo.ico    # Favicon
    └── blogs/       # Blog posts (DD-MM-YYYY.json)
```
